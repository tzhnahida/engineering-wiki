# 2026-06-30 - UWB MIMU 人体运动分析综述

**Yogesh et al. (2023).** *Integrated UWB/MIMU Sensor System for Position Estimation towards an Accurate Analysis of Human Movement: A Technical Review.* Sensors, 23(16), 7277. DOI: `10.3390/s23167277`。

## 做了什么

UWB + 磁惯性测量单元（MIMU）用于人体运动分析的**全链路技术综述**。覆盖硬件选型、标定方法、融合算法（KF/EKF/UKF/PF/因子图）、各应用场景。

## 关键梳理

| 维度 | 覆盖 |
|------|------|
| 硬件 | UWB 芯片选型 + IMU 等级 + 锚点部署策略 |
| 标定 | 天线延迟 + 锚点位置 + IMU 零偏 |
| 算法 | KF → EKF → UKF → PF → 因子图优化 |
| 应用 | 运动分析、康复、体育、室内导航 |
