# MIPI DPI Specification v2.0

## 基本信息

| 属性 | 值 |
|------|-----|
| **标题** | MIPI Alliance Specification for Display Pixel Interface (DPI) |
| **版本** | v2.0 |

| **文件** | `参考/标准/MIPI_DPI_specification_Version_2.0.pdf` |

## 概述

DPI 规范定义了并行像素级显示接口，使用 HSYNC/VSYNC/DE/DOTCLK 时序信号直接传输实时像素流。DPI 是传统 RGB 并行接口的 MIPI 标准化版本，也是 DSI Video Mode 时序模型的原型。
