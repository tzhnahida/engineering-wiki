## 概述

DBI 规范定义了基于微处理器并行总线的显示接口，提供 Type A（Motorola 6800）、Type B（Intel 8080）和 Type C（SPI-like）三种类型。DBI 通常与 DCS 命令集配合使用，适用于低分辨率、有帧缓存的显示模组。
