# 2026-06-24 - Skog 零速检测算法评估

> **论文**: Skog, Händel, Nilsson & Rantakokko (2010) — *Zero-Velocity Detection — An Algorithm Evaluation*, IEEE Transactions on Biomedical Engineering
> **存放**: [参考/论文/Skog 2010 - Zero-Velocity Detection - An Algorithm Evaluation.pdf](参考/论文/Skog 2010 - Zero-Velocity Detection - An Algorithm Evaluation.pdf.md)

## 概述

零速检测（ZUPT）的权威基准论文。提出基于 GLRT（广义似然比检验）的统一框架，推导出 SHOE（Stance Hypothesis Optimal dEtector）检测器。系统评估了四种检测器（角速度能量、加速度移动方差、加速度幅值、SHOE），引入 PSD（Stationarity Detection Probability）替代传统 ROC 作为更适合 ZUPT 的评估指标。结论：角速度能量 ≈ SHOE ≫ 纯加计方案。

## 关键贡献

- GLRT 框架统一推导四种检测器
- SHOE 检测器：融合加计 + 陀螺的最优检测
- PSD 度量：每步态周期至少检测到一次零速的概率
- 陀螺信号比加计信号更适合零速检测
