## 选型指南

| 元件 | 关键参数 | 典型用途 |
|------|---------|---------|
| 电阻 | 阻值、精度、功率 | 偏置、限流、分压 |
| MLCC | 容值、耐压、介质 | 去耦、滤波、储能 |
| 铝电解 | 容值、耐压、ESR、寿命 | 储能、电源输出滤波 |
| 电感 | 感值、DCR、饱和电流 | 升降压转换器、滤波 |
| 磁珠 | 100MHz 阻抗、DCR、额定电流 | 电源轨噪声滤波 |
| 保险丝 | 额定电流、分断能力、时间常数 | 过流保护 |
| PTC | 保持电流、动作电流、Vmax | 可恢复过流保护 |
