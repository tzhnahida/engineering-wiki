# 40MHz Crystal

> 40MHz 石英晶体谐振器。SMD3225，AT 基频。

## 1. 身份与选型

- **频率**: 40MHz (系列支持 8–64MHz)
- **封装**: SMD3225 (3.20×2.50×0.70mm)
- **振动模式**: AT 基频
- **负载电容**: 10pF / 20pF / 定制

## 2. 极限工况

| 参数 | 值 |
|------|-----|
| 存储温度 | -55 – +125°C |
| 驱动电平 | 100μW Max |

## 3. 推荐工作条件

| 参数 | 值 |
|------|-----|
| 频率偏差 (25°C) | ±10ppm / ±20ppm |
| 频率-温度特性 | ±20ppm |
| 工作温度 | -40 – +85°C |
| 老化率 | ±3ppm/年 Max |

## 4. 电气特性

| 参数 | 值 |
|------|-----|
| 负载电容 CL | 10pF / 20pF |
| ESR | 40Ω Max (32–64MHz) |
| 并联电容 C0 | 3pF Max |
