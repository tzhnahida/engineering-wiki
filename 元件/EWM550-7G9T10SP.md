# EWM550-7G9T10SP

亿佰特（Ebyte）国产 UWB 超宽带无线测距定位模组，基于瀚巍创芯 MK8000 SoC。

## 规格

| 参数 | 数值 |
|------|------|
| 芯片 | MKSEMI MK8000 |
| 频段 | CH5 (6.5GHz) / CH9 (8.0GHz) 可切换 |
| 发射功率 | 10 dBm |
| 测距精度 | ±30cm（静态 ±10cm） |
| 测距距离 | 130m CH9 / 200m CH5（空旷） |
| 接口 | UART AT 指令（默认 921600bps） |
| 工作电压 | 1.8–3.6V（典型 3.3V） |
| 发射电流 | 43mA @10dBm |
| 接收电流 | 6.6mA |
| 休眠电流 | 8µA |
| 尺寸 | 14×24mm，邮票孔 |
| 温度 | -40°C ~ +85°C |
| 特性 | 合金屏蔽罩、CH5/CH9 可切换 |
| 价格 | ~¥65 |

## 引脚

| 引脚 | 功能 |
|------|------|
| VCC | 3.3V |
| GND | 地 |
| TXD | UART 发送 |
| RXD | UART 接收 |
| WKP | 低功耗唤醒 |
| SWCLK/SWDIO | 固件烧录（预留） |

## AT 指令

| 指令 | 功能 |
|------|------|
| `AT+ROLE=1/0` | 主机/从机 |
| `AT+PID=1` | 网络 ID |
| `AT+PWR=3` | 发射功率（0-3 档） |
| `AT+CH=5/9` | 切换 CH5/CH9 |
| `AT+PERIOD=100` | 测距周期（ms） |
| `AT+DIST?` | 查询测距值 |
| `AT+ALL` | 查询所有参数 |
