# HDMI Type A
# HDMI Type A

## 1. 身份选型
HDMI Type A (standard) receptacle connector, 19-pin. Full-size HDMI interface.

## 2. 极限工况
- **额定电压**: 5V (per HDMI spec, on +5V pin)
- **额定电流**: 0.5A per HDMI pin (0.05A min for signal pins)
- **工作温度**: −20°C to +85°C
- **插拔寿命**: ≥ 10,000 cycles

## 3. 推荐条件
- **线缆**: Standard HDMI cable with Type A plug
- **焊接**: Reflow (SMT tails) + wave (THT anchors)
- **信号完整性**: Controlled 100Ω differential impedance for TMDS pairs

## 4. 电气特性
- **接触电阻**: ≤ 30mΩ
- **绝缘电阻**: ≥ 100MΩ @ 500V DC
- **耐压**: 500V AC / 1 min
- **特性阻抗**: 100Ω differential (TMDS lines)
- **信号速率**: Up to 18Gbps (HDMI 2.0) / 48Gbps (HDMI 2.1)

## 5. 机械特性
- **尺寸**: — mm (L×W×H — standard 13.9×4.45mm outline)
- **间距**: 0.5mm (contact pitch)
- **插入力**: ≤ 44.1N
- **拔出力**: ≥ 9.8N

## 6. 用途
Audio/video interface — TV, monitor, projector, set-top box, game console.

## 7. 引脚/触点
19 pins — 4 TMDS differential pairs (8 pins), TMDS clock (2 pins), DDC (SCL/SDA), CEC, +5V, Hot Plug Detect, utility/HEAC, and GND. Contact: phosphor bronze, selective gold plating.

## 8. 封装/安装
- **封装**: SMT right-angle with through-hole retention legs
- **安装**: Reflow (signal pins) + wave (anchor pins)
- **卷装**: Tape & reel or tray
