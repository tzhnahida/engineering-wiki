# HDMI Type D
# HDMI Type D

## 1. 身份选型
HDMI Type D (micro HDMI) receptacle connector, 19-pin. Compact version for portable devices.

## 2. 极限工况
- **额定电压**: 5V (per HDMI spec, on +5V pin)
- **额定电流**: 0.5A per HDMI pin
- **工作温度**: −20°C to +85°C
- **插拔寿命**: ≥ 5,000 cycles

## 3. 推荐条件
- **线缆**: Micro HDMI cable with Type D plug
- **焊接**: Reflow soldering
- **信号完整性**: Controlled 100Ω differential impedance (TMDS)

## 4. 电气特性
- **接触电阻**: ≤ 30mΩ
- **绝缘电阻**: ≥ 100MΩ @ 500V DC
- **耐压**: 500V AC / 1 min
- **特性阻抗**: 100Ω differential (TMDS)
- **信号速率**: Up to 18Gbps (HDMI 2.0 compatible)

## 5. 机械特性
- **尺寸**: — mm (L×W×H — approx 6.4×2.8mm, significantly smaller than Type A)
- **间距**: 0.4mm (contact pitch)
- **插入力**: ≤ 44.1N
- **拔出力**: ≥ 9.8N

## 6. 用途
Compact HDMI interface — Raspberry Pi Zero, action cameras, tablets, smartphones.

## 7. 引脚/触点
19 pins — same signal assignment as HDMI Type A in a smaller form factor. Contact: copper alloy, selective gold plating.

## 8. 封装/安装
- **封装**: SMT right-angle
- **安装**: Reflow soldering
- **卷装**: Tape & reel
