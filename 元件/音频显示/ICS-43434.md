# ICS-43434

## 1. 身份选型
- **型号**: ICS-43434（DS-000069 Rev 1.2, 2016-08-29）
- **类型**: 底部进声孔 (bottom port) 数字 MEMS 麦克风，I²S 输出
- **制造商**: TDK InvenSense
- **封装**: 6 端 LGA_CAV，3.50 × 2.65 × 0.98 mm
- **输出**: 标准 I²S，24-bit，二进制补码，MSB 先行——直连 MCU/DSP，**无需音频 codec**
- **工作模式**: 高性能 / 低功耗 (AlwaysOn) / 睡眠，三档由时钟频率自动切换（无模式引脚）
- **兼容性**: 与 ICS-43432 功能兼容，功耗更低、封装更小
- **典型应用**: 可穿戴、智能电视/遥控器、IoT、会议系统、麦克风阵列（±1 dB 灵敏度公差，阵列免校准）

三种模式一览（模式由采样率决定，见 §5）：

| 指标 | 高性能模式 | 低功耗模式 | 睡眠模式 |
|------|-----------|-----------|---------|
| 灵敏度 | −26 dB FS ±1 dB | −26 dB FS ±1 dB | — |
| SNR | 65 dBA | 64 dBA | — |
| AOP | 120 dB SPL | 120 dB SPL | — |
| 电流 | 490 µA | 230 µA | 12 µA |
| 采样率 fS | 23 – 51.6 kHz | 6.25 – 18.75 kHz | < 3.125 kHz |

> [!tip] 三个指标其实是一回事
> EIN = 94 − SNR：29 dB SPL 的等效输入噪声就是"麦克风自身噪声折算成多响的声音"。动态范围 = AOP − EIN = 120 − 29 ≈ 91 dB。而灵敏度 −26 dB FS 恰好等于 −(AOP − 94)：94 dB SPL 输出 −26 dB FS，声压每升 1 dB 输出升 1 dB，到 120 dB SPL 时输出正好触顶 0 dB FS——灵敏度是围绕 AOP 设计出来的，不是越高越好。

## 5. 接口/频率特性
- **数据格式**: 标准 I²S——24-bit/通道，二进制补码，MSB 先行，**MSB 相对 WS 边沿延迟 1 个 SCK**（这是 I²S 与左对齐格式的区别，老页面误记为"左对齐"）
- **帧结构**: 每个 WS 立体声帧**必须恰好 64 个 SCK**（每半帧 32 SCK 槽装 24-bit 数据）；输出完 LSB 后 SD 立即进入三态，把总线让给另一颗麦克风
- **声道选择**: LR 引脚硬件选择——LR=0 输出在左声道（跟随 WS 下降沿），LR=1 输出在右声道（跟随 WS 上升沿）
- **模式切换（全靠时钟频率，无控制寄存器）**:
  - fS 23 – 51.6 kHz → 高性能模式
  - fS 6.25 – 18.75 kHz → 低功耗 AlwaysOn 模式（关键词唤醒、环境声监测）
  - fS < 3.125 kHz（即 SCK 低于约 200 kHz = 64 × 3.125 kHz）→ 睡眠模式，SD 高阻，12 µA
- **模式切换时序**: 入睡 < 1 ms；唤醒后经 32,768 个 SCK 周期开始出数（3.072 MHz 时 10.7 ms；2.4 MHz 时 13.7 ms），20 ms 内灵敏度收敛到最终值 ±1 dB；低功耗 ↔ 高性能切换 < 10 ms
- **多麦克风同步**: 共享同一组 SCK/WS 的多颗麦克风在 **WS 下降沿同步采样**，天然相位对齐；配合 ±1 dB 灵敏度公差，阵列/波束成形无需逐只校准（参考 AN-1140）
- **频率响应**: 60 Hz – 20 kHz 宽带；下限由 24 Hz 固定高通决定，上限由 0.417 × fS 抽取滤波通带决定

> [!warning] 采样率禁区
> 18.75 – 23 kHz 和 3.125 – 6.25 kHz 是模式定义的空隙，手册未定义此区间行为。选 fS 时避开这两段（如不要用 22.05 kHz 采样 CD 半率音频）。

### 信号链：从声波到 I²S 码流
片内完整集成五级：**MEMS 声学膜 → 信号调理 → ΔΣ ADC → 抽取/抗混叠滤波 → I²S 串口**（外加电源管理）。ADC 是单比特高阶 ΔΣ 调制器，以远高于音频带宽的过采样率运行——噪声整形把量化噪声推到音频带外换取带内动态范围，再由低通抽取滤波器把带外噪声滤除并降到目标采样率。前级另有一个 24 Hz 固定转角的高通，去除直流偏置和超低频（风噪、门震）成分。

与"模拟麦 + 外部 ADC/codec"方案对比，这条链的意义在于：
1. **模拟信号不出封装**——麦克风到 MCU 之间全是数字线，长走线、过排线不再拾取 EMI/串扰；
2. **省掉 codec 与偏置电路**——I²S 直进 MCU/DSP 的 I²S 外设；
3. **PSR 高达 −99 dB FS**（217 Hz 方波测试，模拟 DC-DC 纹波最恶劣的场景），可以直接用数字电源轨供电而不需 LDO 净化（去耦仍必须做，见 §8）；
4. **群延迟仅 2/fS（≈42 µs @48 kHz）且确定**，多只麦克风间延迟一致，利于波束成形。

### 灵敏度 −26 dB FS 到底是什么
数字麦克风灵敏度以 dB FS（相对数字满量程）标定：施加 1 kHz、94 dB SPL（= 1 Pa）声压，输出数字信号**峰值**比满量程码低 26 dB。注意 0 dB FS 正弦波的定义是"峰值恰好触及满量程码"——所以 −26 dB FS 描述的是峰值余量，**不能**想当然换算成"RMS 为 −29 dB FS"（这是手册专门澄清的常见误解）。工程意义：94 dB SPL 之上还有 26 dB 声学余量，恰好在 AOP = 120 dB SPL 处用完；超过 120 dB SPL 后 THD 超 10%，波形削顶。

### 双麦立体声接法（Figure 8）
两颗 ICS-43434 的 SCK、WS、SD 三线直接并联到主控同一组 I²S 引脚，一颗 LR 接地（左）、一颗 LR 接 VDD（右）。各自在自己的半帧驱动 SD、其余时间三态，时分共享数据线。**SD 线需接 100 kΩ 下拉电阻**，在所有麦克风都三态的间隙把总线泄放到确定电平；若泄放速度不够（长走线、高容性负载）可换 10 kΩ。SD 三态时自身负载约 6 pF，SD 驱动能力上限 85 pF——估算总线上"走线 + 对侧麦克风 + MCU 输入"的总电容不要超限。

AlwaysOn 用法：一颗以 16 kHz 低功耗模式常听（230 µA 做关键词检测），另一颗断电或睡眠——被禁用的麦克风 SD 呈高阻，不拖累在听的那颗。

## 8. 封装
- **封装类型**: 6-Terminal LGA_CAV（带腔体的芯片阵列小外形无引线）
- **外形尺寸**: 3.50 × 2.65 × 0.98 mm
- **进声方式**: Bottom port——声孔在封装底面，经 PCB 开孔拾音
- **订购**: −40 ~ +85 °C，13 英寸编带，10,000 只/盘；评估板 EV_ICS-43434-FX（柔性板）
- **工艺兼容**: Sn/Pb 与无铅回流均可，RoHS/WEEE 合规

### 声孔与 PCB 布局（声学器件的布局就是性能）
- **PCB 声孔直径 ≥ 0.5 mm**，且必须大于麦克风封装自身声孔直径——孔小了等于给声波加阻塞
- **焊盘 1:1** 按封装焊盘尺寸落板（Figure 13）；钢网开窗避开声孔（Figure 14），**锡膏绝不能进声孔**
- PCB 厚度不影响性能；刚性板、柔性板均可。**柔性板 + 胶层直贴外壳**是手册推荐做法：声孔四周形成可靠密封，同时声学通路最短（通路越短，腔体谐振越远离音频带，高频响应越干净）
- 整机装配时声孔到外壳进声孔之间需用密封圈/胶圈隔离，防止侧漏与灰尘液体侵入（详见 AN-1124 密封应用笔记）

### 回流焊与操作（MEMS 膜片怕气流和超声）
- 回流曲线按手册 Table 7：无铅峰值 260 °C（+0/−5 °C），液相线 217 °C 以上 ~50 s，升温 ≤1.25 °C/s，降温 ≤3 °C/s；兼容 J-STD-020
- 贴片吸嘴只能接触**顶面盖板**，禁止真空吸底面、禁止向声孔吹气或抽气，禁止过大贴装压力
- **洗板水不得接触声孔；禁止超声清洗和吹扫工艺**——超声能直接打坏 MEMS 膜片

### 电源去耦（手册给到布局级细节）
0.1 µF 陶瓷电容（X7R 或更好）接在 Pin 5 (VDD) 与 GND 之间，尽量贴近器件：
- 电容两端连线尽可能短，**走线保持单层、不打过孔**
- 电容与电源/地引脚等距放置；做不到等距时**偏向电源引脚一侧**
- 到地平面的热连接放在电容远离器件的一侧（Figure 12）
