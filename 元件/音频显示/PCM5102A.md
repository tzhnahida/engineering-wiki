# PCM5102A

## 1. 身份选型
- **型号**: PCM5102A（PCM510xA 系列最高档；车规版 PCM5102A-Q1，AEC-Q100）
- **类型**: 立体声音频 DAC（ΔΣ 架构 + 电流分段输出），32-bit / 384 kHz
- **制造商**: Texas Instruments（数据手册 ZHCSA60C，2012–2015）
- **封装**: TSSOP-20（PW），无其他封装选项
- **接口**: I²S / 左对齐，16–32 bit，3 线或 4 线连接
- **典型应用**: A/V 接收器、DVD/BD 播放器、HDTV、车载信息娱乐、树莓派类嵌入式 Hi-Fi 输出

**选它的三个理由**（对应手册三大特性）：
1. **无需 MCLK**——内置音频 PLL 以 BCK 为基准在内部生成系统时钟，主控只需引出 BCK/LRCK/DIN 三根线；
2. **无需输出耦合电容**——DirectPath™ 负电荷泵产生 −3.3 V 轨，输出以真地为中心（2.1 Vrms, GND center）；
3. **无需 I²C/SPI**——纯硬件引脚配置（FLT/DEMP/XSMT/FMT 四脚拉高拉低即完成全部设置）。

### 系列内差异
| 型号 | 动态范围 | SNR | THD+N (−1 dBFS) |
|------|------|------|------|
| **PCM5102A** | **112 dB** | **112 dB** | **−93 dB** |
| PCM5101A | 106 dB | 106 dB | −92 dB |
| PCM5100A | 100 dB | 100 dB | −90 dB |

三者引脚兼容，仅模拟性能分档。同板音频链路的输入侧为 I²S MEMS 麦克风 [[ICS-43434]]，两者可挂在同一 I²S 总线体系下构成采集—回放闭环。

## 5. 接口/频率特性

### 5.1 音频数据格式
- **仅支持 I²S 和左对齐（LJ）两种格式**，由 FMT 脚选择；不支持右对齐、不支持 TDM。
- 数据位宽 16 / 20 / 24 / 32 bit，二进制补码，MSB 在前；DIN 在 BCK 上升沿采样。
- 采样率 fS = 8 ~ 384 kHz（≤192 kHz 全时钟倍数可用；384 kHz 仅 SCK = 64/128 fS）。
- BCK 速率：64、48 或 32 fS（外部 SCK 模式）；PLL 模式仅 32 或 64 fS。

### 5.2 时钟体系（本芯片的核心卖点）
内部数据通路：数据以 fS 进入 → 插值/调制一路升频至 **128 × fS** 送电流分段 DAC。所需的全部高速时钟可来自两种模式：

**模式 A — 外部 SCK（4 线 I²S）**：SCK ≤ 50 MHz，支持 64/128/192/256/384/512/768/1024/1152/1536/2048/3072 fS 共 12 种倍数。片上检测电路自动识别采样率频段（8、16、32–48、88.2–96、176.4–192、384 kHz，容差 ±4%），并自动为数字滤波器、ΔΣ 调制器和负电荷泵配置时钟——**无任何寄存器可设，全自动**。常用组合：

| fS | 常用 SCK（MHz） |
|------|------|
| 44.1 kHz | 11.2896 (256fS)、16.9344 (384fS)、22.5792 (512fS) |
| 48 kHz | 12.288 (256fS)、24.576 (512fS)、49.152 (1024fS) |
| 96 kHz | 24.576 (256fS)、49.152 (512fS) |
| 192 kHz | 24.576 (128fS)、49.152 (256fS) |
| 384 kHz | 24.576 (64fS)、49.152 (128fS) |

**模式 B — 内部 PLL 从 BCK 生成 SCK（3 线 I²S，"无 MCLK"模式）**：上电后器件默认等待外部 SCK；若 **BCK 与 LRCK 已正常运行而 SCK 连续 16 个 LRCK 周期保持低电平**，内部 PLL 自动启动，以 BCK 为基准生成内部 SCK。一旦检测到外部 SCK，PLL 立即被禁用（外部时钟优先）。PLL 模式要求 BCK = 32 或 64 fS：

| fS (kHz) | BCK = 32 fS | BCK = 64 fS |
|------|------|------|
| 8 | — | — |
| 16 | — | 1.024 MHz |
| 32 | 1.024 MHz | 2.048 MHz |
| 44.1 | 1.4112 MHz | 2.8224 MHz |
| 48 | 1.536 MHz | 3.072 MHz |
| 96 | 3.072 MHz | 6.144 MHz |
| 192 | 6.144 MHz | 12.288 MHz |
| 384 | 12.288 MHz | 24.576 MHz |

> [!note] 为什么 3 线模式有价值
> 省掉一条 10~50 MHz 的主时钟走线：布线更容易、EMI 更低，且 ESP32/STM32 等主控无需专门输出 MCLK。8 kHz 采样率无法用 PLL 模式（表中无有效 BCK），需外部 SCK。
> SCK 悬空不可靠——判定条件是"SCK 为地电平"，**未用的 SCK 应接地**而非悬空。

**同步容错**：LRCK 与 SCK 相位漂移超过 ±5 SCK，或 LRCK/BCK 关系错误超过 4 个 LRCK 周期时，器件在一个采样周期内切换到片上振荡器重新初始化，输出强制到双极零点，直至重新同步——这就是"时钟错误不爆音"的机制。

### 5.3 数字插值滤波器（FLT 脚选择）
| 参数 | Normal（FLT=0，FIR） | Low Latency（FLT=1，IIR） |
|------|------|------|
| 通带 | 0 ~ 0.45 fS | 0 ~ 0.47 fS |
| 阻带起点 | 0.55 fS | 0.55 fS |
| 阻带衰减 | −60 dB | −52 dB |
| 通带纹波 | ±0.02 dB | ±0.0001 dB |
| 群延迟 | 20 tS（48 kHz 时 ≈ 417 µs） | 3.5 tS（48 kHz 时 ≈ 73 µs） |

- 纯放音选 Normal（阻带衰减更深，镜像抑制更好）；有唇音同步或实时监听需求（如游戏、乐器监听、与视频对齐）选 Low Latency，延迟缩短近 6 倍，代价是阻带衰减少 8 dB。
- 两种滤波器均按 fS 自动切换 ×8/×4/×2/×1 插值级数（8 ~ 384 kHz 全覆盖）。

> [!warning] 手册内部数值不一致
> 规格表（§8.5）给 Normal 延迟 20 tS，而插值滤波器详表（Table 4–6）给群延迟 22 tS；Low Latency 通带规格表写 0.47 fS、详表写 0.45 fS。做延迟预算时按较保守值（22 tS）留裕量。

## 6. 核心功能

![[_llm/raw/assets/datasheets/pcm5102a/pcm5102a_p14_fig1.jpg|620]]
*内部架构（§9.3）：I²S 接收 → 插值滤波 → ΔΣ DAC → 负电荷泵实现真地输出*

### 6.1 信号链：插值 → ΔΣ 调制 → 电流分段 DAC
1. **音频接口**收取 I²S 数据（fS 速率）；
2. **8× 插值滤波器**把采样率升到中间倍率并滤除镜像；
3. **32-bit ΔΣ 调制器**把高位宽数据整形为高速低位流，量化噪声被推到带外，最终数据率达 128 × fS；
4. **电流分段 DAC**（TI 最新一代 advanced segment 架构）把低位流转成电流——分段结构是它对时钟抖动耐受度高、带外噪声比传统开关电容 DAC 低 20 dB（100 kHz~3 MHz 实测）的关键；
5. **I/V 转换 + 模拟静音 + 线路驱动器**输出 2.1 Vrms 单端信号。片上已含模拟低通重建，外部仅需一个简单 RC。

### 6.2 负电荷泵与"真地"输出（DirectPath™）
传统单电源 DAC 输出以 VDD/2 为中心，必须串大容量耦合电容隔直，带来低频滚降、开机充电爆音和外部静音电路。PCM5102A 用**负电荷泵**从 CPVDD (3.3 V) 产生 **−3.3 V 内部负轨**（VNEG 脚），使输出级工作在 ±3.3 V 双电源下、静态电位恰为 0 V：
- CAPP (pin 2) ↔ CAPM (pin 4) 之间接 **2.2 µF 飞跨电容**；
- VNEG (pin 5) 对地接 **2.2 µF 去耦电容**；
- 电荷泵开关时钟由采样率检测电路自动设定，与音频时钟同源，避免拍频。

结果：**输出直接耦合，无隔直电容**，直流精度好到双极零点误差仅 ±1 mV，且每通道可驱动低至 1 kΩ 的负载——优于一般电荷泵类线路驱动器。

### 6.3 静音体系（防爆音的三层设计）
- **XSMT 软静音**（详见 §7）：数字域每个采样周期 −1 dB 斜降，104 个采样从 0 dBFS 到 −∞，随后模拟静音；反向解除同样 104 采样斜升。完整静音过程 150 tS + 0.2 ms（48 kHz 约 3.3 ms，192 kHz 快 4 倍）。
- **零数据检测**：左右声道同时连续 1024 个 LRCK（48 kHz 时约 21 ms）为零 → 自动进入模拟全静音，静音态 SNR 典型 123 dB（"120 dB 静音 SNR"卖点即此）。
- **时钟错误检测**：数据时钟出错时切到内部振荡器，从最后有效值数字衰减后硬静音到地，再进待机——拔线/断流不爆音。

### 6.4 电源模式与上下电
- **Standby**：检测到时钟错误/停止即自动进入，DAC 与线路驱动器断电（总功耗 ~2 mW）。
- **Power-Down**：BCK 与 LRCK 持续低电平超过 1 s 自动进入，额外关闭负电荷泵与基准（2 mW 典型）。时钟恢复（或满足 PLL 启动条件）后自动上电，无需任何命令。
- **上电复位（POR）**：电源越过阈值（3.3 V 系统 ~2.8 V；1.8 V DVDD 系统为 AVDD/CPVDD ~2.8 V 且 DVDD ~1.5 V）后内部复位约 4 ms 释放；有效数据流过所选滤波器群延迟后输出开始斜升。
- **推荐掉电顺序**（防关机爆音，二选一）：① 断电前 ≥ 150 tS + 0.2 ms 先拉低 XSMT；② 断电前 ≥ 3 ms 先停 I²S 时钟。

### 6.5 XSMT 作欠压保护输入（UVP 模式，仅 DVDD = 3.3 V）
XSMT 不只是静音脚，还能通过电阻分压监测系统母线（如 12 V）实现**断电预静音**：
- XSMT 电平在 ≥ 6 ms 内从高缓降 → 进入外部 UVP 模式：降到 **2 V**（典型 2.2 V）开始软静音；降到 **1.2 V**（典型 0.9 V）强制模拟静音并关断 DAC。
- XSMT 在 ≤ 20 ns 内跳变 → 解释为数字静音命令，软静音后进待机。
- 手册示例：12 V 母线经 7.25 kΩ / 2.75 kΩ 分压，正常时 XSMT = 3.3 V；SMPS 掉电时分压电压先于 3.3 V LDO 跌落，芯片提前静音。

> [!warning] XSMT 输入范围为 −0.3 V ~ DVDD + 0.3 V。分压比必须保证母线正向浪涌不会把 XSMT 抬过 3.6 V（12 V 系统尖峰 > 14.4 V 即越限），否则可能损坏器件。不用 UVP 时 XSMT 可直连 AVDD。

### 外围元件（典型 3.3 V 单电源应用，Fig 33）
- 每个电源脚（CPVDD/AVDD/DVDD）：0.1 µF + 10 µF 就近去耦；LDOO：0.1 µF。
- 飞跨电容 2.2 µF（CAPP–CAPM）、VNEG 2.2 µF。
- 输出滤波：每通道串 **470 Ω** + 对地 **2.2 nF**（fc ≈ 154 kHz），滤除残余带外噪声后送后级——性能规格即在此负载条件（10 kΩ）下测得。
- 后级接什么：功放/有源音箱/ADC 的线路输入（≥1 kΩ）均可直连；**不适合直推 16~32 Ω 耳机**（低于 1 kΩ 负载下限），需外加耳放。

## 8. 封装
- **封装类型**: TSSOP-20（TI 代号 PW），JEDEC MO-153
- **外形尺寸**: 本体 6.4–6.6 × 4.3–4.5 mm，高度 ≤ 1.2 mm
- **引脚间距**: 0.65 mm
- **湿敏等级**: MSL 1，峰值回流 260 °C
- **供货形式**: 编带 2000 只/盘（PCM5102APWR）或管装 70 只（PCM5102APW）

### PCB 布局要点（§12）
- 可用整板共地（AGND 与 DGND 直接相连），无需分割地平面——但各地电位差须 < 0.2 V。
- **分区原则**：数字时钟/接口走线（BCK/LRCK/DIN/SCK）远离 OUTL/OUTR 模拟输出，避免高速时钟回流耦合进模拟区；参考布局中数字脚（11–20）朝主控、模拟脚（1–10）朝输出侧，天然左右分区。
- 所有去耦电容和电荷泵电容尽量贴近芯片引脚放置（飞跨电容回路电流大，优先级最高）。
- 顶层走信号，底层整片铺地作回流面。

> [!note] 与寄存器可编程的姊妹型号（PCM512x/514x/5242，支持软件模式、任意 1–50 MHz 非音频时钟经 PLL 变主模式）不同，PCM5102A 是纯硬件从模式器件：SCK 只接受音频整数倍频，芯片永远是时钟从机。选型时若需要芯片作 I²S 主机或非标时钟，应升级到 PCM512x 系列。
