# SMAJ Series

## 1. 身份选型

400W 表面贴装 TVS 二极管，DO-214AC (SMA) 封装。击穿电压范围 5～70V。峰值功率 400W (10/1000μs)。适用于中低能量瞬态及 ESD 防护场景，如信号线、I/O 端口、通信接口。

## 2. 极限工况

- **峰值脉冲功率 (PPPM)**: 400W (10/1000μs 波形)
- **正向浪涌电流 (IFSM)**: —
- **ESD 能力**: IEC 61000-4-2 (按具体型号确认)
- **工作温度**: -55 至 +150°C
- **存储温度**: -55 至 +150°C

## 3. 推荐工作条件

- **VRWM**: 约 VBR(min) × 0.8
- **工作温度**: -55 至 +150°C

## 4. 功耗热特性

- **峰值功率**: 400W @10/1000μs
- **热阻**: —
- **工作结温**: -55 至 +150°C

> SMAJ 为同系列最小功率等级 (400W)，介于 SMF (200W) 与 SMBJ (600W) 之间。

## 5. IO/电气特性

| 参数 | 说明 |
|------|------|
| VBR (击穿电压) | 5 ～ 170V，按型号后缀选择 |
| VRWM (反向工作电压) | 通常为 VBR × 0.8 |
| 漏电流 (IR) | — |
| 钳位电压 (VC) | 取决于 IPP 和 VBR |
| 响应时间 | < 1ps |
| 方向 | 单向 / 双向 (CA 后缀) |

## 6. 核心功能

- **中低能量瞬态防护**: 400W 浪涌功率，适合次级保护层级。
- **快速响应**: < 1ps 动作速度。
- **双向/单向**: CA 后缀双向适合 AC 耦合信号线。

> 参见 [[电源电子/电源保护机制|电源保护机制]] 了解多级防护架构。

## 7. 引脚

DO-214AC (SMA) 标准 2 引脚。

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | K (Cathode) | 阴极 / 被保护线 |
| 2 | A (Anode) | 阳极 / GND |

双向 TVS (CA 后缀) 无极性。

## 8. 封装

- **DO-214AC (SMA)**: 4.5×2.8mm，表面贴装
- 与 SMB (SMBJ) 封装更小
