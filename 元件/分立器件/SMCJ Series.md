# SMCJ Series

## 1. 身份选型

1500W 表面贴装 TVS 二极管，DO-214AB (SMC) 封装。击穿电压范围 5～70V。峰值功率 1500W (10/1000μs)。适用于高能量瞬态防护场景，如电源输入端、工业总线及汽车系统。参见 [电源保护机制](../../电源电子/电源保护机制.md)。

## 2. 极限工况

- **峰值脉冲功率 (PPPM)**: 1500W (10/1000μs 波形)
- **正向浪涌电流 (IFSM)**: —
- **ESD 能力**: IEC 61000-4-2 (按具体型号确认)
- **工作温度**: -55 至 +150°C
- **存储温度**: -55 至 +150°C

## 3. 推荐工作条件

- **VRWM**: 约 VBR(min) × 0.8
- **工作温度**: -55 至 +150°C
- **瞬态脉冲宽度**: ≤1ms

## 4. 功耗热特性

- **峰值功率**: 1500W @10/1000μs
- **热阻**: —
- **工作结温**: -55 至 +150°C

> SMCJ 是 SMBJ (600W) 的 2.5 倍功率等级，适合更高能量的浪涌防护。

## 5. IO/电气特性

| 参数 | 说明 |
|------|------|
| VBR (击穿电压) | 5 ～ 170V，按型号后缀选择 |
| VRWM (反向工作电压) | 通常为 VBR × 0.8 |
| 漏电流 (IR) | — |
| 钳位电压 (VC) | 取决于 IPP 和 VBR |
| 响应时间 | < 1ps |
| 方向 | 单向 / 双向 (CA 后缀) |

## 6. 核心功能

- **高能量瞬态防护**: 1500W 峰值功率，胜任电源入口级浪涌保护。
- **快速响应**: < 1ps 动作速度。
- **双向/单向**: CA 后缀双向适合 AC 线路。

## 7. 引脚

DO-214AB (SMC) 标准 2 引脚。

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | K (Cathode) | 阴极 / 被保护线 |
| 2 | A (Anode) | 阳极 / GND |

双向 TVS (CA 后缀) 无极性。

## 8. 封装

- **DO-214AB (SMC)**: 7.0×6.0mm，表面贴装
- 比 SMB 封装大，允许更高功率
