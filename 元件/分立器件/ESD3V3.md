# ESD3V3

## 1. 身份选型

3.3V 单向 ESD 防护二极管，SOD-323 封装。专为 3.3V 信号/电源轨的 ESD 浪涌防护设计。高电容特性（需确认具体型号）。

## 2. 极限工况

- **ESD 能力**: —（按具体型号确认 IEC 61000-4-2 等级）
- **峰值脉冲功率**: —
- **工作温度**: -55 至 +150°C
- **存储温度**: -55 至 +150°C

## 3. 推荐工作条件

- **VRWM**: 3.3V
- **工作电压**: 3.3V 轨或 3.3V 信号线
- **工作温度**: -55 至 +150°C

## 4. 功耗热特性

—

## 5. IO/电气特性

| 参数 | 说明 |
|------|------|
| VRWM | 3.3V |
| VBR (击穿电压) | — |
| 漏电流 (IR) | — |
| 钳位电压 (VC) | — |
| 电容 (C) | 高电容（相对 USBLC6 等低容型号） |
| 方向 | 单向 |

> 电容特性需注意：若用于高速信号线（如 I2C、SPI），高电容可能影响信号完整性。

## 6. 核心功能

- **3.3V 系统 ESD 防护**: 为 3.3V 供电轨和 GPIO 提供 ESD 泄放路径。
- **单向保护**: 适合 DC 电压轨（正极对地钳位）。

> 参见 [电源保护机制](../../电源电子/电源保护机制.md) 了解 ESD 防护在整体保护中的角色。

## 7. 引脚

SOD-323 标准 2 引脚。

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | K (Cathode) | 阴极 / 接被保护线路 |
| 2 | A (Anode) | 阳极 / 接 GND |

## 8. 封装

- **SOD-323**: 1.7×1.3mm，紧凑表面贴装
- 极低高度，适合空间受限设计
