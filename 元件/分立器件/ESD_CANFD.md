# ESD_CANFD

## 1. 身份选型

CAN FD 总线 ESD 防护二极管。专为高速 CAN FD 总线设计的低电容 ESD 保护器件。保持信号完整性的同时提供 IEC 61000-4-2 ESD 防护。

## 2. 极限工况

- **ESD 能力**: —（按具体型号确认 IEC 61000-4-2 等级）
- **峰值脉冲功率**: —
- **工作温度**: -40 至 +125°C（汽车级典型）
- **存储温度**: -55 至 +150°C

## 3. 推荐工作条件

- **工作电压**: 符合 CAN/CAN FD 总线共模电压范围
- **总线速率**: 最高 5Mbps+ (CAN FD)
- **工作温度**: -40 至 +125°C

## 5. IO/电气特性

| 参数 | 说明 |
|------|------|
| VRWM | — |
| VBR (击穿电压) | — |
| 漏电流 (IR) | — |
| 钳位电压 (VC) | — |
| **线间电容** | **超低**（确保 CAN FD 高速信号完整性） |
| 通道数 | 2 路（CANH / CANL）|

> 关键参数为低线间电容——CAN FD 速率可达 5Mbps 以上，过高的电容会劣化信号边沿。


## 6. 核心功能

- **CAN FD 总线 ESD 防护**: 保护 CAN 收发器（如 TJA1044）免受 ESD 和瞬态浪涌损伤。
- **低电容设计**: 超低电容保证 CAN FD 高速通信的边沿斜率不受影响。
- **双通道**: 同时保护 CANH 和 CANL 两条总线。

> 参见 [电源保护机制](../../知识/电源电子/电源保护机制.md) 了解系统级保护的层次关系。与 [TJA1044](../接口存储/TJA1044.md) 配合使用时，ESD 器件应尽量靠近总线连接器放置。
