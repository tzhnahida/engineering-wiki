# USBLC6-2SC6

## 1. 身份选型

USB 专用 ESD 保护二极管，2 路单向通道。SOT-23-6L 封装。IEC 61000-4-2 Level 4: ±12kV 接触 / ±17kV 空气放电。峰值脉冲功率 70W (8/20μs)。工作电压 5V。适用于 USB 2.0/3.0、HDMI 1.3/1.4、SATA、DVI、PCIe 等高速数据接口。
## 2. 极限工况

- **ESD 接触放电**: ±12kV (IEC 61000-4-2 Level 4)
- **ESD 空气放电**: ±17kV (IEC 61000-4-2 Level 4)
- **峰值脉冲电流**: 8A (8/20μs)
- **峰值脉冲功率**: 70W (8/20μs)
- **工作温度**: -40 至 +125°C
- **存储温度**: -55 至 +150°C

## 3. 推荐工作条件

- **工作电压**: ≤5V
- **每线工作电流**: 无持续电流（仅瞬态保护）
- **工作温度**: -40 至 +125°C

## 6. 核心功能

[📷 _llm/raw/assets/datasheets/usblc6/usblc6_p4_fig2.jpg|560]
*典型应用：USB 2.0 数据线 ESD 保护接法*


- **高速数据线 ESD 防护**: ±17kV 空气放电能力，远超 USB 规范要求
- **超低电容**: 典型 3pF 线间电容，对 USB 2.0 (480Mbps) 和 HDMI (Gbps) 信号无影响
- **2 路独立保护**: 同时保护 D+/D- 两条差分线，或任意两路信号
- **5V 工作电压**: 兼容 USB VBUS 和 5V 逻辑电平
> 参见 [电源保护机制](电源电子/电源保护机制.md) 了解 ESD 防护在整体保护架构中的定位。
