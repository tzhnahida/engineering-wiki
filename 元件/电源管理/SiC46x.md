# SiC46x
> Vishay microBUCK 系列。4.5–60V、2A–10A、COT 控制、98% 峰值效率。MLP55-27L。

## 1. 身份与选型

| 型号 | 电流 |
|------|------|
| SiC461 | 10A |
| SiC462 | 6A |
| SiC463 | 4A |
| SiC464 | 2A |
