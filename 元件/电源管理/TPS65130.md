# TPS65130
> TI 双路正负输出 DC/DC 转换器：一颗芯片同时给出正压 Boost（最高 +15V）与负压反相 Buck-Boost（最低 −15V），输入 2.7~5.5V，VQFN-24。典型用途：LCD/AMOLED/CCD 偏压、运放与传感器双极性供电。TPS65131 为 2A 开关电流限版本（TPS65130 为 0.8A）。

## 1. 身份与选型

| 项目 | 内容 |
|------|------|
| 型号全称 | TPS65130 / TPS65131（数据手册 ZHCSQG4E，2004 首发、2022 年 4 月 Rev E） |
| 核心功能 | 双路独立输出：正压 Boost + 负压反相 Buck-Boost，共用时钟与基准 |
| 输出能力 | VPOS 最高 +15V（效率至 89%）；VNEG 最低 −15V（效率至 81%）；组合效率约 85% |
| 典型输出电流 | 200~500mA 量级（取决于输入/输出压比与电流限版本） |
| 关键卖点 | 两路独立使能/独立 PSM、外部 PMOS 电池断开控制（BSW）、关断电流典型 0.2µA |
| 选型分界 | TPS65130：开关电流限 0.8A（典型 800mA）；TPS65131：2A（典型 1950mA），两者引脚兼容 |
| 车规 | 本体非车规；TPS65131 有 Q100 车规版 **TPS65131-Q1** |

> [!tip] 什么时候选它
> 系统里只有一路 3.3V/5V 或单节锂电，却需要 ±5V~±15V 双极性轨（运放、比较器、差分音频 PA、DAC/ADC 前端、显示偏压）时，用一颗 TPS6513x 替代"Boost + 负压电荷泵/反相器"两颗方案。两路输出电压、电流可完全不对称。

## 6. 核心功能

[📷 _llm/raw/assets/datasheets/tps65130/tps65130_p8_fig1.jpg|600]
*功能方框图：正压 Boost 与负压反相变换两条独立功率级，仅共用振荡器、基准 Vref 与温度监控*

### 6.1 双通道拓扑：一颗芯片、两个独立变换器

- **正通道（Boost）**：INP 为功率级输入，内部 NMOS + 外部电感 L1 + 外部肖特基 D1 升压到 VPOS。经典异步 Boost，基础原理参见 [DC-DC降压转换器](../../知识/电源电子/DC-DC降压转换器.md)（同为电感式开关变换，占空比关系互为对偶）。
- **负通道（反相 Buck-Boost）**：INN 为功率级输入，开关导通时 L2 从 INN 充电，关断时电感电流经 D2 把 OUTN 拉向负压，得到 VNEG。开关应力为 VIN + |VNEG|（对应 OUTN-INN 差分 24V 极限）。
- 两路仅共享时钟和基准，控制环、使能、PSM、软启动全部独立，因此输出电压与负载可以完全**不对称**（例如 +9V/−4.5V）。

### 6.2 控制架构：定频 PWM + 多重前馈 + 自学习校正

两路均为固定频率 PWM。控制器采用**多重前馈**拓扑：同时监测输入电压、输出电压和开关管压降（即电流信息），工况变化直接改写占空比，不必等输出电压环慢慢响应；前馈的测量误差由内部**自学习系统**校正，CP/CN 引脚外接电容用于平滑该校正输出，避免输出电压跳变。误差放大器内部补偿，只处理小信号误差——所以外围**不需要**设计补偿网络，只需按推荐值放 CP = 10nF、CN = 4.7nF。

### 6.3 工作模式

| 模式 | 条件 | 行为 |
|------|------|------|
| CCM 定频 PWM | 中重载 | 两路均连续导通；Boost 侧另有内部开关允许电感电流回流输入，**始终保持 CCM** |
| DCM | 轻载（仅负通道） | 反相通道电感电流可断续，控制器自动适配 |
| 省电模式 PSM | PSP/PSN = 高，轻载 | 输出跌破内部阈值才打出一个或数个脉冲，随后休眠；轻载效率显著提高，纹波变大、频率不定 |
| 强制 PWM | PSP/PSN = 低 | 恒定开关频率，纹波小、频谱可预测，轻载效率低 |

> [!tip] PSM 用不用？
> 音频、精密采集等对纹波频谱敏感的负载建议 PSP/PSN 拉低用强制 PWM（频率固定在 1.25~1.5MHz，滤波好设计）；电池供电待机场景拉高 PSP/PSN 换轻载效率。两路可以一路 PSM、一路强制 PWM 混用。

### 6.4 独立使能与上电时序

- ENP 控制正压、ENN 控制负压，高电平使能（0V 禁止，VIN 使能）。
- 使能后先启动该通道所需内部电路，随后进入软启动。
- 两个 EN 都拉低 → 器件进入关断模式，全部内部电路关闭，仅 0.2µA（典型）流入 VIN。
- 时序完全由外部决定：可先正后负、先负后正或同时。运放系统常见做法是**先建立负轨再建立正轨**（或同时），避免输出级在单轨期间输出不确定电平——本器件用两个 GPIO 即可实现任意顺序。

### 6.5 软启动

每次使能时，开关电流限从零在约 **1ms** 内爬升到额定值（800mA / 1950mA）。目的：限制启动瞬间从电池抽取的浪涌电流，避免电池内阻上的压降拖垮同电源域的其它负载、或误触电流限。注意：若软启动期间带满载启动，输入电流会高于稳态，电源选型要覆盖。

### 6.6 负载断开与 BSW 电池开关

- **负通道**：关断时内部 PMOS 断开，无残留直流路径，不会放电。
- **正通道**：L1 + D1 天然构成"输入→输出"直流通路，芯片内部**无法**切断——关断后 VPOS 仍约等于 VIN 且会持续漏电。解决办法：BSW 引脚驱动外部 PMOS（Q1）串在电池与 L1 之间；ENP 拉高时 BSW 输出低电平使 Q1 导通，ENP 拉低时 Q1 关断实现完全断开。PMOS 体二极管方向必须指向电池侧，否则体二极管仍会导通。
- 不需要断开功能时 BSW **悬空**即可。

### 6.7 保护机制 → [电源保护机制](../../知识/电源电子/电源保护机制.md)

| 保护 | 阈值 | 动作 |
|------|------|------|
| 逐周期电流限 | 800mA / 1950mA（典型） | 限制开关峰值电流 |
| 过压保护 OVP | 反馈电压超标称 +5%（典型） | 对应通道立即关断，保护后级 |
| 欠压闭锁 UVLO | VIN < 2.35V（典型） | 双路同时关断，控制电路部分保持活动（区别于关断模式） |
| 热关断 | TJ > 150°C，迟滞 5°C | 双路关断，降温后自恢复 |

VIN 分区行为：VIN > 2.7V 全性能；VUVLO < VIN < 2.7V 功能不关闭但性能不保证；VIN < VUVLO 全部关闭、耗流最小。

### 7.1 输出电压设定（反馈分压）

**正通道**：分压接 FBP，FBP 调节点 = 1.213V。分压电流应取 5~10µA（约为 FBP 偏置电流 50nA 的 100 倍），即 R2 < 200kΩ。

$$R_1 = R_2\left(\frac{V_{POS}}{V_{ref}} - 1\right), \quad V_{ref} = 1.213\,\mathrm{V}$$

**负通道**：分压接 FBN，但下端**不接地，接 VREF**。FBN 调节点为 0V，R4 上压降恒为 1.213V。分压电流同样取 ≥5µA，R4 < 200kΩ。

$$R_3 = R_4 \cdot \frac{|V_{NEG}|}{V_{ref}}$$

手册给出的整定值：

| 目标输出 | 上电阻 | 下电阻 | 前馈电容 |
|----------|--------|--------|----------|
| VPOS = +10.5V | R1 = 1MΩ | R2 = 130kΩ | C9 = 6.8pF |
| VPOS = +15V | R1 = 975kΩ | R2 = 85.8kΩ | — |
| VNEG = −10V | R3 = 1MΩ | R4 = 121.2kΩ | C10 = 7.5pF |
| VNEG = −15V | R3 = 1.3MΩ | R4 = 104.8kΩ | — |

> [!warning] 负通道分压的经典坑
> FBN 分压的参考点是 VREF（1.213V）而不是 GND——画原理图时把 R4 下端接地，输出会整体偏移且不受控。VREF 引脚必须挂 220nF 旁路电容（C8），它同时是负反馈网络的基准源。

### 7.2 电感选择

先估峰值电流（0.64 系数已折算约 80% 效率与占空比余量），保证峰值低于所选器件的开关电流限（TPS65130：800mA；TPS65131：1950mA）：

$$I_{L(P)} = \frac{I_{POS} \cdot V_{POS}}{V_I \times 0.64} \qquad I_{L(N)} = \frac{I_{NEG} \cdot (V_I + |V_{NEG}|)}{V_I \times 0.64}$$

再按纹波电流 ΔIL ≈ 20% × IL 计算电感量（f 取 1.25MHz）：

$$L_1 = \frac{V_I \,(V_{POS} - V_I)}{\Delta I_{L(P)} \cdot f \cdot V_{POS}} \qquad L_2 = \frac{V_I \cdot |V_{NEG}|}{\Delta I_{L(N)} \cdot f \cdot (V_I + |V_{NEG}|)}$$

手册示例（VI = 3.3V）：两路算得均约 5.1µH → 实际选 **4.7µH**。器件针对 **3.3~6.8µH** 优化；更大感量需另做稳定性分析。纹波取小（<20%）可降电感损耗、输出纹波与 EMI，但环路响应变慢、电感体积成本上升。推荐料：Wurth 7447789 系列 / 744031 系列（TPS65130）、EPCOS B82462-G4 系列（TPS65131）、TDK VLF3010/VLF4012、Cooper SD12。

### 7.3 输出电容

先按允许纹波 ΔV（忽略 ESR）算最小容量：

$$C_{4min} = \frac{I_{POS} \,(V_{POS} - V_I)}{f \cdot \Delta V_{POS} \cdot V_{POS}} \qquad C_{5min} = \frac{I_{NEG} \cdot |V_{NEG}|}{f \cdot \Delta V_{NEG} \,(|V_{NEG}| + V_I)}$$

纹波目标 10mV 时约需 12µF。再叠加 ESR 分量 ΔV_ESR = I × R_ESR：10mΩ 级陶瓷电容额外贡献约 2mV。手册 BOM 每路输出用 **4 × 4.7µF / 25V / X7R 并联**（原理图标注 22µF 即此并联组合）——25V 耐压 + 并联既覆盖 ±15V 输出，又缓解陶瓷电容直流偏压容量缩水。负载瞬态会带来额外跌落/过冲（电感限制电流爬升速率），实际容量建议比计算值再放大。

### 7.4 输入电容与模拟供电滤波

- INP、INN 各就近放 ≥4.7µF 陶瓷输入电容（C1、C2）；用钽电容时需并联 100nF 陶瓷。
- **模拟供电滤波（Rev E 新增章节）**：VIN（控制电路）与 INN（功率级）之间加 RC 或 LC 滤波去噪。常规用 R7 = 100Ω（最小）+ C3 = 0.1µF；若输入电压上升沿很快（**压摆率 ≥ 275mV/µs**，如热插拔到已建立的 5V 轨），R7 换成铁氧体磁珠（如 Murata BLM18KG101TN1，DCR 尽量小，额定电流取最大输入电流的 1.5~2 倍），减小 INN 与 VIN 信号间延迟。

### 7.5 整流二极管

两路各需一只肖特基（降低损耗）：正向电流额定 ≥ 该路最大输出电流，反向耐压覆盖输出电压。手册参考料 Onsemi MBRM120（1A / 20V）。注意两路 IPOSmax / INEGmax 可能不同，可分开选型。

### 7.6 外部 PMOS（Q1）

仅在需要关断时彻底断开正输出/电池时使用（见 6.6）。选型要点：VGS、VGD 耐压覆盖输入电压范围；漏极电流额定 ≥ 最大输入电流；校验安全工作区。参考料 Vishay Si2323DS（P 沟道，12V / 4A）。栅极经 BSW 驱动（BSW 内阻 27kΩ）。

### 7.7 环路稳定：前馈电容与补偿电容

- 反馈上电阻并联前馈电容加速环路：C9 = 6.8µs / R1，C10 = 7.5µs / R3（常数为时间量纲，代入 R1 = R3 = 1MΩ 得 6.8pF / 7.5pF，与 BOM 一致）。
- 前馈电容会把输出噪声耦合进环路，可与其串联 10k~100kΩ 电阻限带宽——阻值越大，耦合噪声越小。
- CP = 10nF（C6）、CN = 4.7nF（C7）为自学习校正的外部阻尼电容，按推荐值放置即可，环路补偿本体在片内。

### 7.8 PCB 布局：两个功率回路分开处理

- 本器件有**两个独立的高 di/dt 回路**：Boost 回路（C1→L1→INP 开关→D1→C4→PGND）和反相回路（C2→INN 开关→L2/OUTN→D2→C5→PGND）。两个回路各自面积最小化，输入/输出电容、电感、二极管全部紧贴 IC。
- 主电流路径与功率地走线**宽而短**；用大面积 PGND 平面；AGND 引脚就近接入 PGND 平面；裸焊盘焊接并连 PGND（兼顾散热与接地）。
- 反馈分压电阻紧贴引脚放置：正通道贴 FBP，负通道贴 VREF 引脚——负反馈网络以 VREF 为基准，VREF 走线被干扰等于基准被干扰。
- 布局不当的症状：线性/负载调整率变差、地弹与输出偏移、稳定性问题、EMI 超标、效率下降（手册 9.1 原话归纳）。
- 热设计：RθJA = 34.1°C/W 的前提是裸焊盘经由过孔阵列进入内层铜箔；TA = 85°C 时允许耗散约 1.17W，双路满载 ±15V 时要核算损耗是否越限。

### 7.9 典型应用场景

- **显示偏压**：4~17 英寸 LCD / AMOLED 源极驱动正负轨（笔记本、显示器、游戏机、电梯 HMI、医疗、EPOS）。
- **通用双极性电源**：测试测量、数据采集、DAC/ADC 基准与前端、差分音频功放、工厂自动化 IO 模块、差分运放/比较器供电——单电池或 3.3V/5V 轨直接产生 ±5~±15V。
- 输入源：单节锂电（2.7~4.2V）、3 节碱性/NiMH、或任意 2.7~5.5V 稳压轨。
