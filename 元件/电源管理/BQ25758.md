# BQ25758
> TI I2C 控制型**双向**四开关升降压（Buck-Boost）**控制器**，外置 4 颗 NFET。4.2–60V 输入，3.3–60V 输出，20A 级，面向 USB PD EPR / 充电桩 / 显示器。QFN-36。
> 数据手册：ZHCSSI5B（2022-12 首发，2024-03 Rev B）；英文原版 SLUSET4。

## 1. 身份与选型

| 项目 | 内容 |
|------|------|
| 型号 | BQ25758RRVR（RRV = VQFN-36，R = 3000 只卷带） |
| 功能 | I2C 控制、双向同步降压/升压 **控制器**（NFET 驱动器 + 集成环路补偿） |
| 卖点 | USB PD EPR 双向（正向 CC-CV 输出 / 反向源出），20A@5mΩ 检流，16-bit ADC 遥测 |
| I2C 地址 | 0x6B（7 位），标准/快速/超快速模式，最高 1Mb/s |
| 器件识别 | REG0x3D：PART_NUM = 100b（BQ25758），DEV_REV = 010b |
| 工作温度 | 订购表 -40～85°C（推荐 TA -40～105°C，TJ ≤125°C） |

**控制器 vs 集成方案的意义**：BQ25758 只含栅极驱动器、环路补偿、检流放大器和保护逻辑，功率 MOSFET、电感、检流电阻全部外置。因此功率等级由外围决定——手册参考设计做到 48V/5A（240W），电流环支持到 20A（5mΩ 检流），远超集成式 buck-boost（如 [TPS55288](TPS55288.md)，内置 FET 上限约 6A 级）。代价是 BOM 更多、布局要求高。它属于 TI 充电器（BQ）家族，可作电池充电控制器使用，但**芯片本身不含充电状态机**（见 §6.3 坑位说明）。

### 6.1 四开关 Buck-Boost 架构

单电感 + 两个半桥：Q1/Q2（HIDRV1/LODRV1，SW1，降压腿）+ Q3/Q4（LODRV2/HIDRV2，SW2，升压腿）。控制器按 VAC 与 VOUT 关系在三种模式间**无缝连续切换**（表 6-2）：

| 模式 | Q1（buck HS） | Q2（buck LS） | Q3（boost LS） | Q4（boost HS） |
|------|------|------|------|------|
| Buck（VAC≫VOUT） | fSW 开关 | fSW 开关 | 关断 | 常通 |
| Buck-Boost（VAC≈VOUT） | SW1/SW2 交错开关 | 交错 | 交错 | 交错 |
| Boost（VAC≪VOUT） | 常通 | 关断 | fSW 开关 | fSW 开关 |

- **环路补偿全集成**（"无补偿转换器"），软启动内置——外部无需 RC 补偿网络 → [环路补偿](环路补偿.md)
- **升压占空比限制**：boost 模式下 HS FET 每周期强制导通 225ns 以转移电感能量。500kHz 时最大占空比 = 1 − 0.225µs/2µs = 88.75%，5V 输入理想最大输出约 44V；降低 fSW 可放宽该限制
- **轻载 PFM**：EN_PFM=1 时轻载降频提效 → [轻载效率模式](轻载效率模式.md)；检测到 FSW_SYNC 外部时钟时该位自动清零（可再手动置 1 强制 PFM）
- **旁路模式**：EN_BYPASS=1 时 Q1、Q4 常通，输入直通输出（VOUT=VAC），消除开关与磁芯损耗；旁路中 IOUT 超过 IOUT_REG 即自动退出并进入高阻态（置 IBAT_OCP_STAT + INT），恢复需清 EN_HIZ。旁路栅驱电压可选 10V/7V（PWRPATH_REDUCE_VDRV）
- **栅极驱动**：RON 3.4Ω / ROFF 1.0Ω；死区默认 45ns，可编程 45/75/105/135ns（REG0x3C）；驱动强度四档可调（REG0x3B）用于 EMI/效率折衷；BTST 自举刷新阈值 3.1V

### 6.2 双向功率流（USB PD EPR）

- **正向**：输入→输出，四环路仲裁——VOUT_REG（CV）、IOUT_REG（CC）、VAC_DPM（输入欠压回缩）、IAC_DPM（输入限流），实际工作点由最先触及的环路决定
- **反向**：EN_REV=1，输出（电池）→输入端，VAC 被调节至 VAC_REV（3.3–60V/20mV 步进，POR 5V），电流由 IAC_REV 限制；即 USB PD 反向供电（Source 角色）
- 反向峰值电流限制 IBAT_REV：20/15/10/5A 四档（REG0x62）
- 反向保护：REV_OVP（超调节目标停开关，自动恢复）、REV_UVP（低于 SYSREV_UV——VAC_REV 目标的 80% 或固定 3.3V——则清 EN_REV 退出反向）
- EN_CONV_FAST_TRANSIENT（POR=1）加快瞬态响应；升压模式存在右半平面零点 RHPz = VIN/(IIN·2πL)，UGBW 应 ≤ RHPz/3，必要时增大 Cload（手册例：5V→7V/5A/10µH，RHPz≈11.4kHz，快速瞬态模式需 Cload ≥674µF；关闭后 ≥80µF 即可）

> [!warning] 没有充电状态机——这是"控制器"，不是"充电器"
> 手册中**不存在**预充电、涓流、终止（termination）、再充电（recharge）阈值，也没有安全定时器。CHARGE_STAT[2:0] 仅报告 000b=不开关 / 011b=CC 模式 / 100~110b=CV 模式。完整的锂电充电流程（预充→CC→CV→终止→再充、JEITA 降额）必须由主机借助 16-bit ADC + I2C 在软件中实现。若需硬件充电状态机，选同家族 BQ25756。原页所写"NVDC 功率路径"亦不存在——本器件无系统/电池功率路径管理。

### 6.3 输入侧动态功率管理（DPM）与太阳能输入

- **输入限流**（IAC_DPM 寄存器 vs IIN 引脚，取**较低**者）：输入电流触及限值时转换器自动回缩输出电流，允许用小容量适配器。IIN 引脚：IAC_MAX = KILIM/RIIN（@2mΩ）；非调节状态下 VIIN 正比于输入电流，可作模拟监测（IAC = KILIM×VIIN/(RIIN×2V)）
- **输入电压调节**（VAC_DPM 寄存器 vs ACUV 引脚 VACUV_DPM=1.21V，取**较高**者）：输入电压跌至限值时回缩电流、维持输入电压——手册明确指出适用于**太阳能电池板**等高内阻源。注意这是"定电压钳制"，MPPT 寻优算法需主机通过 ADC 读 VAC/IAC 并动态改写 VAC_DPM 实现
- 电流回缩到零仍拉不住电压、VACUV 跌破 VREF_ACUV（1.1V）时停止开关
- **VAC 工作窗口**：三电阻分压（VAC→ACUV→ACOV→PGND），VACOV = 1.2V×(RAC1+RAC2+RAC3)/RAC3，VACUV = 1.1V×(RAC1+RAC2+RAC3)/(RAC2+RAC3)；顶部 RAC1 推荐 1MΩ 以减小漏电流。不用时 ACUV 接 VAC、ACOV 接 PGND（回落到内部窗口 4.2–66V）
- **高阻态模式**：EN_HIZ=1 或 IIN 引脚 >3.7V；停开关、关 REGN，负载由电池供电

### 6.4 状态机与主机交互

- **默认模式 / 主机模式**：POR 后处于默认模式（看门狗已到期状态）——自主输出 5V、限流由 IOUT 引脚电阻决定，无主机也能工作。任一 I2C 写入即进入主机模式并启动看门狗；喂狗写 WD_RST=1，或 WATCHDOG=00b 禁用。看门狗到期回默认模式并复位选定寄存器（EN_CHG 复位值可由 EN_CHG_BIT_RESET_BEHAVIOR 选 0 或 1）
- **使能逻辑**：正向开关 = EN_CHG 位（POR=1）且 CE 引脚拉低；CE 不许悬空
- **INT 事件源**：输入源有效/移除、进入 IAC_DPM/VAC_DPM 调节、看门狗到期、TS_STAT 变化、TSHUT、任意 *_STAT 上升沿；每源三位（STAT 现状 / FLAG 读清锁存 / MASK 屏蔽）
- **PG 引脚**（低有效）四条件：VVAC_OK<VVAC<66V、VACUV>1.1V、VACOV<1.2V、非高阻态

### 6.5 保护体系 → [电源保护机制](电源保护机制.md)

| 保护 | 阈值 | 行为 |
|------|------|------|
| VAC_OVP | ACOV 引脚编程 或 内部 66V（退出 63V） | 停开关，PG 拉高，自动恢复；去抖 100µs 进 / 12ms 出 |
| VAC_UVP | ACUV 引脚编程 或 内部 VVAC_OK 4.2V（跌落 3.5V 禁用） | 同上 |
| VBAT_OV / IBAT_OCP | 输出过压 / 电池过流 | 置 STAT/FLAG + INT |
| DRV_SUP 窗口 | UVP 3.7V（上升允许）/ OVP 典型 13.2V（12.8–13.6V） | 停开关，恢复后自动回原模式 |
| REGN UVP | VREGN_OK 3.55V | 停开关，自动恢复 |
| TSHUT | 150°C / 135°C 恢复 | 正反向均有效 |
| 旁路 OCP | IBYPASS_OCP：IOUT_REG=5A 时 4.5–5.5A | 退旁路 + 强制 HIZ |
| 多级过载（可选） | ILIM2 = 150%/200% × 限值，持续 tOVLD 25/50ms；ILIM3 = 20A 绝对上限（@5mΩ），窗口 1ms；新过载间隔 tMAX 100ms | EN_OVLD/EN_OVLD_3L 使能，服务瞬时大负载（如电机启动） |

- **TS 热敏电阻**：默认**禁用**（EN_TS=1 开启），REGN→TS→PGND 分压 + 103AT-2 10kΩ NTC；越窗即**暂停功率变换**，去抖 25ms；TS_STAT 报告 正常/温/凉/冷/热 五态。反向模式阈值可编程：BHOT 55/60/65°C 或禁用，BCOLD -10/-20°C
- > [!warning] 无 JEITA 降额：REG0x1C（TS 区域行为控制）各位全部 RESERVED（注释残留 EN_JEITA 字样但未开放）。TS 只做"越限暂停"，充电电流/电压随温度分档降额需主机软件实现。

### 6.6 16-bit ADC 遥测

- 通道：IAC、IOUT、VAC、VOUT、TS；单次（ADC_DONE_STAT/FLAG + INT）或连续转换；运行平均可选（ADC_AVG）
- 分辨率/速度三档：15 位 24ms、14 位 12ms、13 位 6ms（每通道）
- LSB：VAC/VOUT 2mV（0–65534mV）；IOUT 2mA @5mΩ（±20A，二进制补码）；IAC 2mA @2mΩ（±50A；寄存器表 @5mΩ 时 0.8mA/±20A）；TS 0.098% × REGN
- ADC 与故障无关地持续运行；仅电池供电时 ADC 会自动开 REGN 抬高静态电流——省电需占空比运行或关闭

### 6.8 外围设计要点（手册 §7 参考设计：450kHz，5A，USB PD EPR）

- **开关频率**：FSW_SYNC 下拉电阻编程（式 3）；常用值 200kΩ→200kHz、133kΩ→250kHz、100kΩ→300kHz、57.1kΩ→450kHz（参考设计取 56.9kΩ/0.1%）、50kΩ→500kHz、40kΩ→600kHz；亦可同步外部时钟 200–600kHz（VIH 1.3V，脉宽 ≥80ns）
- **电感**：纹波电流按最大电感电流的 20–40% 取；ISAT ≥ IL + IRIPPLE/2；buck 纹波 = VAC·D(1−D)/(fSW·L)，boost 纹波 = VAC·D/(fSW·L)，D≈0.5 时最恶劣；正反向都要核算
- **MOSFET 选型**（参考 BOM）：48V EPR → 80V/6.2mΩ（SiR880BDP）×4 + 10µH/22mΩ；仅 28V EPR → 40V/3.5mΩ（AONS66408）×4 + 4.7µH/22mΩ——耐压按最高母线电压留 ~1.6 倍裕量
- **检流电阻**：ROUT_SNS 固定 5mΩ；RAC_SNS 默认 2mΩ（寄存器满量程 50A/125mA 步进），EPR 应用推荐 5mΩ 换取 50mA 步进精度——此时 IAC_DPM 寄存器写值与 KILIM 均需乘 2/5 换算；两对检流引脚各接 0.47µF 差分滤波 + 0.1µF 共模滤波
- **MODE 引脚**（仅 POR 时采样一次，±1–2% 电阻）：≤3.0kΩ = buck-boost 自动检测电感（2.2–15µH，L/DCR ≥1260µs）；仅降压按电感值选 4.7kΩ(3.3µH)/6.04kΩ(4.7µH)/8.2kΩ(5.6µH)/10.5kΩ(6.8µH)/13.7kΩ(8.2µH)/17.4kΩ(10µH)/≥27kΩ(15µH)。仅降压必须**移除 Q3/Q4**（SW2 侧），且启用反向前先置 EN_CHG=0
- **输入/输出电容**：陶瓷 + 大容量组合，陶瓷贴近半桥；大容量在检流电阻两侧均匀分布；boost 稳定性要求 COUT ≥80µF（关闭快速瞬态时）
- **PCB 布局**（§9 优先级）：① buck 输入环路（Q1/Q2+CIN）与 boost 输出环路（Q3/Q4+COUT）面积最小、同层放置；② 检流采用开尔文接法、差分走线远离开关节点；③ 栅驱线 ≥20mil、最短化，HIDRVx 与 SWx 成对相邻布线，LODRV 下方铺地回流；④ BTST/DRV_SUP 电容紧贴 IC；⑤ 过孔按 1oz/10mil 每孔 1–2A 估算；⑥ 散热焊盘过孔星形连接 PGND 与功率地平面；⑦ ACUV 分压顶端与 VO_SNS 直接引至目标调节点，避开开关节点

> [!tip] 上电即用：POR 默认 5V 输出 + IOUT 引脚硬件限流，无 MCU 也能作固定 5V 转换器跑起来；主机只需在 40s 看门狗窗口内接管。

### 6.9 主机侧实现锂电充电流程（设计建议，非手册内容）

芯片只提供 CC/CV 两个硬件环路，完整充电状态机由主机软件基于寄存器组合出来：

1. **初始化**：喂狗或禁用看门狗 → 写 VOUT_REG = 满充电压（如 4S 磷酸铁锂 14.6V）→ 写 IOUT_REG = 快充电流 → EN_CHG=1 + CE 拉低
2. **预充**：读 VOUT_ADC，若电池电压低于预充阈值，先写小 IOUT_REG（如 0.1C），越过阈值后再切快充电流——芯片无自动预充档
3. **CC→CV**：由硬件环路自动完成，主机轮询 CHARGE_STAT[2:0]（011b=CC，100–110b=CV）观察切换
4. **终止**：CV 阶段轮询 IOUT_ADC，电流跌破终止阈值（如 C/20）时置 EN_CHG=0 或拉高 CE 停止——芯片不会自动终止
5. **再充**：停充后监测 VOUT_ADC，跌破再充阈值（如满充 −3%）重新使能
6. **温度分档（软件 JEITA）**：读 TS_ADC（REGN 百分比）换算温度，分档改写 IOUT_REG/VOUT_REG；TS 硬件仅在越窗时暂停变换，充当最后防线
7. **安全定时**：主机自建计时器（芯片无安全定时器），超时强制停充并告警
