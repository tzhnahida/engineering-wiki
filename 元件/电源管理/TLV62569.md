# TLV62569
> TI 2A 同步降压转换器，1.5MHz 自适应关断时间（DCS-Control 架构），95% 峰值效率，SOT 封装。配合 TPS54302 实现 5.5V→28V 全电压覆盖。

> [!note] 原元件页多处参数缺失或与手册不符，本次重写以手册 SLVSDG1C（Rev C）为准。主要修正：VIN 极限耐压、ESD 等级、FB 基准、热阻、软启动时间、开关频率描述（非固定）。

## 6. 核心功能

### 6.1 DCS-Control 架构详解（核心价值）

手册原文对控制方案的表述是 **"adaptive off-time with peak current control"（自适应关断时间 + 峰值电流控制）**——与 TI TPS62 系列宣传的 DCS-Control 同族，均属**滞环 + 纹波直接调制的准 COT 变体**。它与传统固定频率电流模式 PWM 的**本质区别**如下：

**固定频率电流模式 PWM**：
- 内部振荡器固定时钟 → 逐周期强制开启
- 误差放大器（EA）输出电压作为峰值电流的基准
- 开关频率恒定的代价是 **EA 带宽限制瞬态响应速度**；输出跌落 → EA 积分上升 → 允许更高峰值电流——经过至少几个周期

**DCS-Control（TLV62569）**：
- 没有用于决定开关时刻的固定时钟，也没有误差放大器（无 EA 主导的慢速环）
- **VFB 直接连到比较器**：输出跌 → FB 低于 VREF → **立即触发导通脉冲**，不等待时钟、不等待 EA 爬升——这是瞬态响应快的根本原因
- 导通脉冲由峰值电流检测终止（电感电流到达电流环设定值时关高侧），**关断时间则由一个简单电路按 VIN/VOUT 比自适应设定**——手册称这使开关频率"相对恒定"（约 1.5MHz），但本质上无时钟锁定，**频率随 VIN、VOUT、负载漂移**（手册 Figure 12/13 给出频率 vs 负载 / vs 输入电压曲线，纵轴跨 0~2.5MHz）
- **纹波直接参与调制**：输出纹波本身就是 PWM 斜坡信号。反馈分压电阻上的前馈电容 C3（6.8pF，§7.1）是为**不衰减地传递输出纹波到 FB 引脚**，保证比较器有足够信号幅度

| 对比维度 | 固定频率电流模式 PWM | DCS-Control（准 COT） |
|---------|------------------|---------------------|
| 开关频率决定方式 | 时钟固定 | **自适应关断时间**，随 VIN/VOUT/负载漂移 |
| 瞬态响应决定因素 | EA 带宽（需数个周期） | **比较器逐周期直接触发** |
| 频率确定性 | 高（可精确设计输入滤波） | 低（Figure 12/13：随负载与 VIN 明显变化） |
| 环路补偿 | 需要 Type II/III | **不需要外部补偿**（纹波幅度需足够） |
| EMI 确定性 | 好 | 略差（频率不固定，无法精准滤波器设计） |

> 这也是为什么 TLV62569 的输出电容范围相当宽（4.7-100µF 均可工作，见 §7.1 选型表）——因为没有补偿网络约束。这不是"精度差"，而是架构特性。

### 6.2 PSM 省电模式

→ [轻载效率模式](../../知识/电源电子/轻载效率模式.md)

**进入判据**：负载降低到电感电流进入**断续模式（DCM）**——零电流检测（ZCD）电路检测到低侧电流过零时进入。手册未给出显式阈值电流，但 CCM/DCM 边界可推算：$I_{OUT} < \Delta I_L / 2$。以典型应用（5V→1.8V、L=2.2µH、1.5MHz）代入 §7.1 纹波公式得 ΔIL≈0.35A，即约 **0.17A 以下**进入 PSM（推算值，随 VIN/VOUT/L 变化）。这与 TPS54302 Eco-mode（固定 500mA 峰值电流阈值）的判据机制不同。

**进入后**：
- 低侧 ZCD 启用（电感电流过零即关低侧，阻止输出电容经 SW 反向放电）
- 高侧仅在 VFB 跌到 VREF 以下时打脉冲，脉冲间隔随负载减小拉长
- 开关频率随负载降低而降低，开关损耗与静态损耗同步收缩，35µA IQ 成为主导
- PWM ↔ PSM 之间**无缝自动切换**，无模式切换毛刺

**退出判据**：负载回升使电感电流重新连续（回到 CCM），自动切回 1.5MHz PWM。

**PSM 代价**：轻载时输出电压会**略高于标称值**（手册未量化），纹波为低频突发包络。手册对策："increasing the output capacitor" 可减轻此效应。

### 6.3 100% 占空比模式

输入电压掉到接近输出电压时，高侧管**持续导通**、低侧管保持关断——不再切换。此时输出电压等于 $V_{IN} - I_{OUT}\times(R_{DS(ON)} + DCR_L)$。

$$V_{IN(MIN)} = V_{OUT} + I_{OUT} \times (R_{DS(ON)} + R_L)$$

其中 $R_{DS(ON)}=100\,m\Omega$（高侧），$R_L$ 为电感 DCR。这一特性对电池供电穿越放电末期很有价值。

### 6.4 软启动与预偏置

使能后内部缓升电路控制输出电压单调爬升。DBV 封装软启动时间 800µs，PDDC/DRL/PDRL 为 900µs（典型值）。

**预偏置安全启动**：若输出已有残压（大电容未放完 / 上游电源反灌），TLV62569 检测到 FB 电压已高于软启动电压则**不拉低侧管**，从现有的偏置电压开始平滑爬升——不会出现低侧管短时导通倒灌大电流的典型危险。

### 6.5 保护

- **OCP**：峰值电流检测到 ILIM → 关高侧、开低侧进入自适应关断时间泄流 → 下周期再检测。手册**未描述打嗝模式**，持续过载时逐周期限流常开，温升靠热关断保护
- **热关断**：TJ > 150°C 关断，降至 130°C 恢复
- **UVLO**：VIN < 2.45V（典型）停止开关，恢复约 2.55V

### 7.1 外围 LC 选型流程

**① FB 分压设定 VOUT**（1% 电阻，R2 推荐 ≤200kΩ）：

$$V_{OUT} = V_{FB} \times \left(1 + \frac{R1}{R2}\right),\qquad V_{FB}=0.6\,V$$

R2=100kΩ（示例），$R1 = R2 \times (V_{OUT}/V_{FB} - 1)$。R2 越大待机功耗越小，但噪声敏感度增加。

**② 电感选择**——先估算纹波电流 $\Delta I_L$，推荐纹波系数 K=0.2~0.4：

$$\Delta I_L = \frac{V_{OUT}(V_{IN} - V_{OUT})}{V_{IN} \times f_{SW} \times L},\qquad
I_{L,MAX} = I_{OUT,MAX} + \frac{\Delta I_L}{2}$$

其中 $f_{SW}\approx 1.5\,MHz$（实际随工况偏移，见 §6.1）。电感饱和电流需比 $I_{L,MAX}$ 高 20-30%，且**至少 ≥ 限流值 3A（DBV/PDDC）/ 2.5A（DRL/PDRL）**。注意手册脚注：电感有效感量可偏 **+20%/-30%**，电容有效容值可偏 **+20%/-50%**（直流偏置降额），计算须以降额后有效值代入。

**LC 组合验证表**（手册 Table 4 提炼）：

- 支持的电感值为 **1.0µH 与 2.2µH** 两档；**2.2µH + 22µF 是各输出电压下的标准推荐组合**（手册标注 "standard value and recommended for most applications"）
- COUT 候选值：4.7 / 10 / 22 / 2×22 / 100 µF；**输出电压越高，可用的 COUT 组合越多**（0.6~1.2V 段仅 22µF 一档通过验证，≥1.8V 段 1.0µH 与 2.2µH 均有多档通过）
- 手册 §8.2.2.5 给出总原则：**COUT 设计工作范围 10µF~47µF**；表外组合需自行仿真 + 实测验证

**③ 输出电容**——陶瓷 X7R/X5R（高频阻抗与温度稳定性好），低 ESR 带来低输出纹波。手册未给出 COUT 计算公式（稳定性由 Table 4 组合保证），如需按瞬态跌落粗估可用通用式（**非手册公式**）：

$$C_{OUT} \gtrsim \frac{\Delta I_{LOAD} \times t_{resp}}{\Delta V_{OUT}}$$

其中 $t_{resp}$ 为环路响应时间，DCS-Control 架构下约为 1~2 个开关周期。

**④ 前馈电容 C3**——并联在 R1（上分压电阻）两端，加速纹波传递到 FB，改善瞬态。C3≈**6.8pF**（当 R2=100kΩ）。前馈电容不是强制件，手册称"optional, 6.8 pF if it is needed"。不加 C3 时环路稳定但瞬态响应较慢；加 C3 后带宽拓宽，纹波传递改善——本质上是补偿 COT 架构在低 ESR 陶瓷输出电容下纹波幅度不足的问题。

> [!tip] C3=6.8pF 的推荐值是针对 R2=100kΩ 的。若 R2≠100kΩ，按 $\tau = C3 \times (R1 \parallel R2)$ 时间常数不变的原则调整——但 6.8pF 是典型用法的最优值，贸然增大可能引入额外噪声耦合。

**⑤ 输入电容 CIN**——低 ESR 陶瓷，4.7µF 起（典型 10µF/0805/X7R）。手册示例：C1=4.7µF（Murata GRM21BR71A475KA73L）。

**⑥ 输入电压范围**：2.5V~5.5V，与 TPS54302 的 4.5-28V 无重叠——这是选型时的第一道分水岭。

### 7.2 PCB 布局要点（手册 §10 提炼）

- **输入电容 CIN 紧贴 VIN-GND**——这是开关环路 di/dt 最大的回路（CIN→VIN→高侧→SW→低侧→GND→CIN），环路面积最小化是 EMI 关键
- **SW 走线短而宽**，远离 FB 反馈走线；避免 FB 检测到 SW 噪声
- **FB 是敏感节点**：走线远离 SW/电感端，FB 分压电阻的 GND 点应单点连接到 IC 的 GND 引脚（开尔文连接）
- **GND 层铺铜散热**：SOT 封装散热高度依赖 PCB 铜箔，多打散热过孔到内层/底层
- **电感靠近 SW 引脚**，输出电容 COUT 靠近电感输出端
