## 定义

**二元函数**：设 $D \subset \mathbb{R}^2$，映射 $f: D \to \mathbb{R}$，记 $z = f(x,y)$，$(x,y) \in D$。

- **定义域** $D$：自变量的取值范围
- **值域** $f(D) = \{z \mid z = f(x,y), (x,y) \in D\}$
- **图形**：空间点集 $\{(x,y,z) \mid z = f(x,y), (x,y) \in D\}$，一般是一张曲面

$n$ 元函数：$u = f(x_1, x_2, \ldots, x_n)$，$(x_1,\ldots,x_n) \in D \subset \mathbb{R}^n$。

## 平面点集基础

### 邻域

点 $P_0(x_0, y_0)$ 的 $\delta$ 邻域：
$$U(P_0, \delta) = \{(x,y) \mid \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta\}$$

**去心邻域** $\overset{\circ}{U}(P_0, \delta)$：不含 $P_0$ 的邻域。

### 点与点集的关系

| 类型 | 条件 |
|------|------|
| **内点** | $\exists U(P) \subset E$ |
| **外点** | $\exists U(P) \cap E = \varnothing$ |
| **边界点** | 任何邻域内既有 $E$ 的点也有非 $E$ 的点 |
| **聚点** | 任何去心邻域内总有 $E$ 的点 |

### 点集分类

| 类型 | 定义 |
|------|------|
| 开集 | 所有点都是内点 |
| 闭集 | 边界 $\subset E$ |
| 连通集 | 任意两点可用属于 $E$ 的折线连接 |
| 区域（开区域） | 连通的开集 |
| 闭区域 | 开区域连同其边界 |
| 有界集 | $\exists r > 0$ 使 $E \subset U(O, r)$ |

## 多元函数的极限

设 $f(x,y)$ 在 $\overset{\circ}{U}(P_0)$ 有定义，若 $\forall \varepsilon > 0$，$\exists \delta > 0$，使当 $0 < \sqrt{(x-x_0)^2+(y-y_0)^2} < \delta$ 时：

$$|f(x,y) - A| < \varepsilon$$

则称 $\lim\limits_{(x,y)\to(x_0,y_0)} f(x,y) = A$。

> [!note] 与一元函数的关键区别
> 二元极限要求沿**任何路径**趋近时极限都存在且相等。只沿某几条路径极限相等不能证明极限存在。

## 多元函数的连续性

若 $\lim\limits_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0,y_0)$，则称 $f$ 在 $P_0$ 连续。

> [!note] 有界闭区域上连续函数的性质
> - **有界性与最大值最小值定理**：在有界闭区域上连续的多元函数必有界，且能取到最大值和最小值
> - **介值定理**：在有界闭区域上连续的多元函数必能取到介于最大值和最小值之间的任何值

## n 维空间 $\mathbb{R}^n$

$\mathbb{R}^n = \{(x_1,\ldots,x_n) \mid x_i \in \mathbb{R}\}$

距离：$\rho(x,y) = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}$

范数：$\|x\| = \sqrt{\sum_{i=1}^n x_i^2}$

极限：$x \to a \iff x_1 \to a_1, \ldots, x_n \to a_n$
