## 定义

在空间取一定点 $O$ 和三个两两垂直的单位向量 $\vec{i}, \vec{j}, \vec{k}$，确定三条以 $O$ 为原点的坐标轴：
- $x$ 轴（横轴）
- $y$ 轴（纵轴）
- $z$ 轴（竖轴）

构成 **Oxyz 坐标系**（或 $[O; \vec{i}, \vec{j}, \vec{k}]$ 坐标系）。

> [!note] 右手规则
> 右手握住 $z$ 轴，四指从 $x$ 轴正向以 $\frac{\pi}{2}$ 转向 $y$ 轴正向，大拇指指向 $z$ 轴正向。

## 坐标分解

任一向量 $\vec{r}$ 可唯一分解为：
$$\vec{r} = \overrightarrow{OM} = x\vec{i} + y\vec{j} + z\vec{k}$$

记作 $\vec{r} = (x, y, z)$，其中 $x\vec{i}, y\vec{j}, z\vec{k}$ 称为沿三个坐标轴的**分向量**。

点 $M$ 坐标记作 $M(x, y, z)$。

## 卦限

三个坐标面（$xOy, yOz, zOx$）将空间分为**八个卦限**，用 Ⅰ~Ⅷ 表示。

| 卦限 | $x$ | $y$ | $z$ |
|------|-----|-----|-----|
| Ⅰ | + | + | + |
| Ⅱ | - | + | + |
| Ⅲ | - | - | + |
| Ⅳ | + | - | + |
| Ⅴ | + | + | - |
| Ⅵ | - | + | - |
| Ⅶ | - | - | - |
| Ⅷ | + | - | - |

## 点的坐标特征

- **坐标面上**：$yOz$ 面 → $x=0$；$zOx$ 面 → $y=0$；$xOy$ 面 → $z=0$
- **坐标轴上**：$x$ 轴 → $y=z=0$；$y$ 轴 → $x=z=0$；$z$ 轴 → $x=y=0$
- **原点**：$x=y=z=0$

## 向量的模

$$|\vec{r}| = \sqrt{x^2 + y^2 + z^2}$$

## 两点间距离

$$|AB| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$$

## 方向角与方向余弦

非零向量 $\vec{r}$ 与三坐标轴的夹角 $\alpha, \beta, \gamma$ 称为**方向角**。

$$(\cos\alpha, \cos\beta, \cos\gamma) = \frac{1}{|\vec{r}|}(x, y, z) = \vec{e}_r$$

满足：$\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1$

## 投影

向量 $\vec{a}$ 在 $u$ 轴上的投影 $\text{Prj}_u\vec{a}$：

性质：
1. $\text{Prj}_u\vec{a} = |\vec{a}|\cos\varphi$（$\varphi$ 为 $\vec{a}$ 与轴的夹角）
2. $\text{Prj}_u(\vec{a}+\vec{b}) = \text{Prj}_u\vec{a} + \text{Prj}_u\vec{b}$
3. $\text{Prj}_u(\lambda\vec{a}) = \lambda\text{Prj}_u\vec{a}$

在直角坐标系中，$\vec{a}$ 的坐标即为其在三坐标轴上的投影：
$$a_x = \text{Prj}_x\vec{a}, \quad a_y = \text{Prj}_y\vec{a}, \quad a_z = \text{Prj}_z\vec{a}$$
