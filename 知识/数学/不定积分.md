## 定义

若 $F'(x)=f(x)$，则 $F(x)$ 是 $f(x)$ 的一个**原函数**。

$f(x)$ 的全体原函数称为 $f(x)$ 的**不定积分**：

$$\int f(x)\mathrm{d}x = F(x) + C$$

> **原函数存在定理**：连续函数必有原函数（见[定积分](定积分.md)中变上限积分）。

**几何意义**：$\int f(x)\mathrm{d}x$ 表示一族互相平行的曲线（积分曲线族），$C$ 不同则上下平移。

## 基本积分表

| 积分 | 结果 |
|------|------|
| $\int k\mathrm{d}x$ | $kx+C$ |
| $\int x^\mu\mathrm{d}x$ | $\frac{x^{\mu+1}}{\mu+1}+C$（$\mu\neq-1$）|
| $\int\frac{1}{x}\mathrm{d}x$ | $\ln|x|+C$ |
| $\int e^x\mathrm{d}x$ | $e^x+C$ |
| $\int a^x\mathrm{d}x$ | $\frac{a^x}{\ln a}+C$ |
| $\int\sin x\mathrm{d}x$ | $-\cos x+C$ |
| $\int\cos x\mathrm{d}x$ | $\sin x+C$ |
| $\int\tan x\mathrm{d}x$ | $-\ln|\cos x|+C$ |
| $\int\cot x\mathrm{d}x$ | $\ln|\sin x|+C$ |
| $\int\sec^2x\mathrm{d}x$ | $\tan x+C$ |
| $\int\csc^2x\mathrm{d}x$ | $-\cot x+C$ |
| $\int\frac{1}{1+x^2}\mathrm{d}x$ | $\arctan x+C$ |
| $\int\frac{1}{\sqrt{1-x^2}}\mathrm{d}x$ | $\arcsin x+C$ |
| $\int\sec x\tan x\mathrm{d}x$ | $\sec x+C$ |

## 第一类换元法（凑微分）

$$\int f[\varphi(x)]\varphi'(x)\mathrm{d}x \xlongequal{u=\varphi(x)} \int f(u)\mathrm{d}u$$

> **核心思想**：被积表达式的一部分恰好是另一部分的导数。

### 常见凑微分形式

| 形式 | 凑法 |
|------|------|
| $\int f(ax+b)\mathrm{d}x$ | $\frac{1}{a}\int f(ax+b)\mathrm{d}(ax+b)$ |
| $\int f(x^n)x^{n-1}\mathrm{d}x$ | $\frac{1}{n}\int f(x^n)\mathrm{d}(x^n)$ |
| $\int f(\sin x)\cos x\mathrm{d}x$ | $\int f(\sin x)\mathrm{d}(\sin x)$ |
| $\int f(e^x)e^x\mathrm{d}x$ | $\int f(e^x)\mathrm{d}(e^x)$ |
| $\int f(\ln x)\frac{1}{x}\mathrm{d}x$ | $\int f(\ln x)\mathrm{d}(\ln x)$ |

**例**：$\int\frac{1}{2x+1}\mathrm{d}x = \frac{1}{2}\ln|2x+1|+C$

**例**：$\int\tan x\mathrm{d}x = \int\frac{\sin x}{\cos x}\mathrm{d}x = -\ln|\cos x|+C$

## 第二类换元法

$$\int f(x)\mathrm{d}x \xlongequal{x=\psi(t)} \int f[\psi(t)]\psi'(t)\mathrm{d}t$$

### 常用代换

| 被积函数含 | 代换 | 理由 |
|-----------|------|------|
| $\sqrt{a^2-x^2}$ | $x=a\sin t$（$t\in[-\frac{\pi}{2},\frac{\pi}{2}]$）| $\sqrt{a^2-x^2}=a\cos t$ |
| $\sqrt{a^2+x^2}$ | $x=a\tan t$（$t\in(-\frac{\pi}{2},\frac{\pi}{2})$）| $\sqrt{a^2+x^2}=a\sec t$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec t$（$t\in(0,\frac{\pi}{2})\cup(\pi,\frac{3\pi}{2})$）| $\sqrt{x^2-a^2}=a\tan t$ |

**例**：$\int\frac{1}{\sqrt{a^2-x^2}}\mathrm{d}x \xlongequal{x=a\sin t} \int\mathrm{d}t = t+C = \arcsin\frac{x}{a}+C$

**倒代换**：当分母次数远高于分子时，令 $x=\frac{1}{t}$。

## 分部积分法

$$\int u\mathrm{d}v = uv - \int v\mathrm{d}u$$

> **口诀**：反对幂指三——按此顺序优先选 $u$（反三角函数、对数、幂、指数、三角）

### 典型模式

| 类型 | $u$ 选择 | 例 |
|------|---------|-----|
| $\int x^n e^x\mathrm{d}x$ | $u=x^n$ | 分部 $n$ 次 |
| $\int x^n\sin x\mathrm{d}x$ | $u=x^n$ | 分部 $n$ 次 |
| $\int e^x\sin x\mathrm{d}x$ | 任意 | **循环**型，解方程 |
| $\int x^n\ln x\mathrm{d}x$ | $u=\ln x$ | $x^n\mathrm{d}x=\mathrm{d}v$ |
| $\int\arctan x\mathrm{d}x$ | $u=\arctan x$ | $\mathrm{d}v=\mathrm{d}x$ |

**例**：$\int x\cos x\mathrm{d}x = x\sin x - \int\sin x\mathrm{d}x = x\sin x + \cos x + C$

**例（循环型）**：$\int e^x\sin x\mathrm{d}x = \frac{e^x}{2}(\sin x-\cos x)+C$

## 有理函数积分

真分式 $\frac{P(x)}{Q(x)}$ 可分解为**部分分式**之和：

$$\frac{P}{Q} = \sum\frac{A}{(x-a)^k} + \sum\frac{Bx+C}{(x^2+px+q)^m}\quad(\Delta=p^2-4q<0)$$

### 步骤

1. 若分子次数 ≥ 分母，先做多项式除法
2. 分母因式分解
3. 待定系数法求 $A,B,C$
4. 逐项积分

## 可化为有理函数的积分

### 三角有理式

$\int R(\sin x,\cos x)\mathrm{d}x$：**万能代换** $t=\tan\frac{x}{2}$

$$\sin x=\frac{2t}{1+t^2},\ \cos x=\frac{1-t^2}{1+t^2},\ \mathrm{d}x=\frac{2}{1+t^2}\mathrm{d}t$$

> 万能代换总能成功，但常较繁琐。优先考虑凑微分。

### 简单无理式

$\int R(x,\sqrt[n]{ax+b})\mathrm{d}x$：令 $t=\sqrt[n]{ax+b}$

$\int R(x,\sqrt[n]{\frac{ax+b}{cx+d}})\mathrm{d}x$：令 $t=\sqrt[n]{\frac{ax+b}{cx+d}}$
