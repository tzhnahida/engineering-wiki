# UWB 室内定位与动捕融合

> Ultra-Wideband 无线电测距 → 绝对 3D 位置观测 → EKF 全局融合 → 补偿 IMU/ZUPT 长期漂移

## UWB 原理

UWB 通过**飞行时间测距（ToF / TWR）** 获得标签→锚点的精确距离（理论 ~10cm LOS），多点距离交会出 3D 绝对位置。

| 参数 | DW1000 系 | MK8000 系（国产） |
|------|:---:|:---:|
| 频段 | CH5 (6.5GHz) | CH5/CH9 (6.5/8.0GHz) |
| LOS 精度 | ~10 cm | ~10–30 cm |
| NLOS 精度 | 1–3 m（人体遮挡） | 0.5–2 m |
| 接口 | SPI（需写驱动） | UART AT 指令 |
| 功率 | ~100mA Tx | 10–43mA Tx |
| 价格/模块 | ¥30–50 | ¥20–80 |

> [!note] CH5 vs CH9
> CH5（6.5GHz）穿人体衰减 ~35dB，CH9（8.0GHz）~42dB。差值 7dB ≈ 5 倍信号强度差。**身体穿戴场景必须优先 CH5。**

## 在动捕系统中的角色

UWB **不是替代任何现有传感器**，而是给 EKF 全局融合提供一个**不漂的绝对位置观测**。

```
IMU 姿态 (高频, 短时精, 长时漂)
ZUPT 位移 (中频, 走路精, 累积漂)
UWB 位置 (低频, 常值噪声, 永不漂) ← 第三条腿
光学位置 (高频, mm 级, 怕遮挡)
```

四条定位链在 EKF 里互相补位：IMU 管姿态和短时位移，ZUPT 管步间，UWB 压长时不漂，光学管精修。

## 级联融合架构

承接 [ESKF](../../姿态解算/误差状态卡尔曼滤波.md) 框架，UWB 在**位置观测层**加入：

```
IMU 六轴 → Madgwick/VQF (姿态 KF) → 四元数
                                          ↓
UWB 测距 → 三边定位 → 位置 KF ← 加速度(去重力) ← 姿态
                                          ↓
                                    3D 绝对位置
```

**Zihajehzadeh 2015** 证明了级联（先姿态后位置）精度接近全局 KF 且计算量可控。**Shi 2023** 验证了 Madgwick + KF(UWB) 在上肢动捕中可行——RMSE 改善 1.2cm。

## 人体遮挡（最大的坑）

身体穿戴场景下，**人体遮挡 NLOS 是 UWB 的主要误差源**，不是时钟漂移也不是环境多径。

| 场景 | 测距误差 |
|------|:---:|
| 标签→锚点 LOS | ~10–20 cm |
| 身体部分遮挡 | 50–150 cm |
| 身体完全遮挡 | 1–3 m |

**De Cock 2023**：用 IMU 朝向 + GMM 误差模型做遮挡检测，p75 误差降 37%。

> [!note] 本系统的独有优势
> 全身 IMU 动捕系统拥有多节点姿态信息，可以精确推断腰→各锚点的视线是否被身体遮挡 → 选择性地信任/排除特定锚点的测距值。这是单 IMU 方案做不到的。

## 锚点部署

4 锚点挂房间四角（高 1.5–2m，俯角覆盖），标签在腰上。锚点坐标需标定（手动量测 ±5cm 即可，自动标定见 [Liu & Cao 2025](../../来源/2026-06-30 - UWB IMU 下肢动捕级联Kalman.md) 因子图法）。

## EKF 观测模型

UWB 三边定位输出 $\mathbf{p}_{\text{uwb}} \in \mathbb{R}^3$，作为 EKF 位置观测：

$$\mathbf{z}_k = \mathbf{p}_{\text{uwb}} = \mathbf{p}_{\text{root}} + \mathbf{n}_{\text{uwb}}$$

其中 $\mathbf{n}_{\text{uwb}} \sim \mathcal{N}(0, \mathbf{R}_{\text{uwb}})$，$\mathbf{R}_{\text{uwb}}$ 根据遮挡检测**自适应缩放**——LOS 时 $\sigma \approx 20\text{cm}$，检测到遮挡时 $\sigma$ 动态放大或排除该锚点。

与 ZUPT 观测共存时，EKF 自动权衡：ZUPT 管 0.1–2s 短时零速约束，UWB 管 >5m 长时不漂。

## 论文创新点定位

1. **分布式人体遮挡检测** — 15 节点全身姿态 → 精确推断腰→锚点 LOS/NLOS，非单 IMU 方案可比
2. **ZUPT-UWB 紧耦合** — ZUPT 短时精度 + UWB 长时不漂，在因子图里统一表示为互补约束
3. **¥1300 系统接入 UWB 仅为 ¥325** — 成本可控，对标 ¥15k 商用系统的传感阵列完整性
